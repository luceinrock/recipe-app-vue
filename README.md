# Frontend with Vue.js

Frontend mockup for an upcoming recipe app built on Spring Boot Java backend.

Demo: https://vue-recipe-app.herokuapp.com/

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
